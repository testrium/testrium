# Features Overview

Testrium covers the full test management lifecycle — from writing test cases to reporting results.

## Core Features

| Feature | Description |
|---|---|
| [Test Case Management](/features/test-cases) | Create, organize, and maintain test cases |
| [Test Run Execution](/features/test-runs) | Execute tests and track results in real time |
| [Bulk Operations](/features/bulk-operations) | Update multiple test results at once |
| [JIRA Integration](/features/jira-integration) | File bugs from failed executions |
| [Test Data Management](/features/test-data) | Environment-specific test data store |
| [Reports & Analytics](/features/reports) | PDF/Excel reports with charts and trends |

## Platform Features

- **Role-Based Access** — Admin and User roles with project-level permissions
- **Dark Mode** — Full dark mode support across the UI
- **Automation API** — REST endpoints for Selenium, RestAssured, and Playwright
- **Bulk Import** — Import test cases from Excel or CSV files
- **Project Management** — Multi-project support with team members
- **Application & Module Hierarchy** — Organize tests by application and module
