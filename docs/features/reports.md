# Reports & Analytics

Generate detailed reports and track quality trends over time.

## Dashboard

The main dashboard shows a real-time summary across all your projects:
- Total test cases
- Active test runs
- Overall pass/fail rate
- Recent activity

## Project Metrics

Drill into any project to see:
- Test case counts by status, priority, and type
- Execution trends over time
- Module-level pass/fail breakdown
- Team member contribution

## Test Run Reports

For any completed test run, generate a detailed report with:
- Summary (total, pass, fail, blocked, skipped)
- Pie chart of results distribution
- Per-module breakdown
- Execution timeline

## Export Formats

| Format | Contents |
|---|---|
| **PDF** | Summary, charts, and execution table — ready to share with stakeholders |
| **Excel** | Full data with all execution details — useful for further analysis |

## Trend Analysis

Track quality over time with trend charts:
- Pass rate per week/month
- Test execution velocity
- Defect discovery rate

## Custom Report Names

Reports can be named before export, making it easy to organize by sprint, release, or test cycle.
