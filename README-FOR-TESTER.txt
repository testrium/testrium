--------------------------------------------------------------------------------
                        TROUBLESHOOTING
--------------------------------------------------------------------------------

Problem: "Java is not installed"
Solution: Install JDK 17+ from https://www.oracle.com/java/technologies/downloads/

Problem: "Maven is not installed"
Solution: Install from https://maven.apache.org/download.cgi

Problem: "Port already in use"
Solution: Run STOP-APPLICATION.bat, wait 10 seconds, try again

Problem: Browser opens but shows error
Solution: Wait 1-2 minutes, then refresh the page
