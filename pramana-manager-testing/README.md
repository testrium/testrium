# Pramana Manager Testing Configuration

Version 1.7.0 - Quick testing setup for different systems

## Quick Start

### Windows
```cmd
setup-windows.bat
```

### Linux/Mac
```bash
chmod +x setup-linux.sh
./setup-linux.sh
```

### Docker
```bash
docker-compose up -d
```

## Configuration Files

- `application-test.properties` - Test database config
- `test-config.json` - Test environment settings
- `docker-compose.yml` - Container setup
- `setup-windows.bat` - Windows setup script
- `setup-linux.sh` - Linux/Mac setup script
